# 📩 Spam Message Detector AI

An AI-powered spam message detector built using Machine Learning and NLP.

## 🚀 Features

- Detects spam messages
- Streamlit web interface
- Machine Learning model
- NLP text processing
- Confidence score prediction

## 🛠 Technologies Used

- Python
- Scikit-learn
- Streamlit
- Pandas
- TF-IDF Vectorizer

## 📊 Model Accuracy

Achieved approximately 97% accuracy on real SMS spam dataset.

```bash
streamlit run app.py
